## 模版简介

该模板（暂且叫 SimpleReport 吧）是基于 [Typst](https://github.com/typst/typst) 编写的模版，用于撰写实验报告和论文。本人撰写的报告最后老师评分都挺高的，强烈安利 Typst！

## 使用说明

### 如何使用

下载 [template.typ](https://github.com/Yang-ZhiHang/report-template/blob/master/template.typ) 到本地，可以参考仓库中的示例 [论文示例](https://github.com/Yang-ZhiHang/report-template/blob/master/examples/paper.typ) [实验示例](https://github.com/Yang-ZhiHang/report-template/blob/master/examples/lab-report.typ) 进行编写。

### 字体

使用前先确认已经安装 Noto Serif CJK SC 和 Times New Roman 字体。
